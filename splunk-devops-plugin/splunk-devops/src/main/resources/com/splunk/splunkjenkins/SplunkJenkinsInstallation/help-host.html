The Splunk Indexer hostname or IP address. Example: "192.168.1.4"

  • For Splunk cloud user, the host name is something like http-inputs-xx.splunkcloud.com or http-inputs-xx.cloud.splunk.com, and HTTP Input Port is 443
  • For Splunk enterprise user, the host name is the indexer host name, and HTTP Input Port is 8088 by default
You need enable "HTTP Event Collector" In Splunk to use the plugin, please checkout HTTP Event Collector
