Important

If this is the first time running this you will need to create a config. Create a storybook-config.json file in your static-storybook folder which contains the following:

{
  "storybook": {
    "blabbr": {
      "db": {
        "user": "DB_USERNAME",
        "pwd": "DB_PASSWORD",
        "host": "DB_HOST"
      },
      "slack": {
        "endPoint": ""
      },
      "ui": {
        "avatar": true
      }
    },
    "versions": {
      "availableVersions": [
        "0.0.1",
        "0.0.2",
        "0.1.2",
        "0.2.5"
      ],
      "hostname": "localhost:8000",
      "localhost": "localhost:9001",
      "regex": "\/([^\/]+?)\/?$"
    }
  }
}

You can skip the blabbr section is you are not using that addon it.
